Lession 1: Introduction

What is HTML?

  • HTML stands for Hypertext Markup Language and is the language webpages are written in.
  • HTML documents are composed of HTML elements.
  • All HTML pages start with a doctype declaration that tells the browser which version of the HTML is being used.
  • The latest version is HTML5.
  • After the doctype declaration, HTML pages then have an html element.
  • Inside the HTML element is a head element where inforamtion about the page goes.
  • After the head element there's a body element where the contents of the page goes.
  • Most HTML elements have opening and closing tags.
  • HTML elements can have attributes that add more information.
  • Some HTML elements do not use a closing tag.
Check out these example of HTML on the W3 School site!

What is CSS?

  • CSS stands for Cascading Style Sheets.
  • CSS controls the presentation of a webpage.
  • CSS uses CSS rules to control the presentation of HTML elements. CSS rules have a selector, properties and values.
  • CSS can be added to a webpage using:
    1. Inline styles - add the style attribute to HTML elements
    2. Internal CSS - puts CSS rules inside a style element placed in the head section of a webpage.
    3. External stylesheets - use a link element in the head section of a webpage. This links to an external .css file that contains CSS rules.
You can find examples of CSS on the W3 School site too!
